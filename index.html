Forest Fire Regrowth Analysis

University Machine Learning (ML) Project

  • Project was developed in Jupyter Notebook using Google Colab
  • Implemented Random Forests (RF) ML Model to Identify Key Variables Contributing to Post-Wildfire Vegetation Regeneration
  • Used Python Pandas Library to perform pre-processing on Airborne Visible/Infrared Imaging Spectrometer (AVIRIS) GIS measurments from the California Rim Fire
  • Trained RF model using scikit-learn and Numpy using Normalized Burn Ratio (NBR) as label
  • Used various vegetation and moisture indexes from AVIRIS data as features
  • By analyzing the frequency of features in top level nodes of the Decision Trees (DT) within the RF model framework provided insight into which variables contribute the most to vegetation recovery
